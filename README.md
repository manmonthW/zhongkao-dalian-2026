## 2026年大连中考志愿核心框架

### 批次结构

```
第一批次：普通高中
  ├── A段：大连十五中（美术）
  ├── B段：提前录取优质高中 + 二十四中金普校区
  ├── C段：指标到校（国家级示范性高中）
  └── D段：一般普通高中（公办+民办）+ 征集志愿

第二批次：职业院校
  ├── 第一次征集志愿
  └── 第二次征集志愿
```

### 三大关键机制

| 机制 | 含义 | 重要性 |
|------|------|--------|
| **指标到校** | 优质高中名额按比例分到初中，降分录取 | ⭐⭐⭐ 普通学生进名校核心通道 |
| **征集志愿** | 未录取考生二次填报机会 | ⭐⭐ 兜底保障 |
| **跨区报考** | 可在大连市内跨区域报考 | ⭐⭐ 扩大选择范围 |
