# 2026年大连中考志愿表填写原则详解

> 综合三份资料：政策文件、志愿表结构、9中2025年录取数据

---

## 一、志愿表完整结构（第一批次 · 普通高中）

### 第一段：提前录取 — 大连市第十五中学
| 志愿栏 | 数量 | 说明 |
|--------|------|------|
| 学校志愿 | 1个 | 设1个统招类志愿，仅限十五中（美术方向） |

**填写原则**：想走美术特长方向才填，不影响后续批次。不想报可空着。

---

### 第二段：省级示范性普通高中 + 二十四中金普校区

这是**最关键的一段**，包含三类志愿：

| 类型 | 志愿数量 | 说明 |
|------|----------|------|
| **统招** | 3个平行学校志愿（A/B/C） | 按分数高低竞争，全市统一分数线 |
| **指标到校** | 2个平行学校志愿（A/B） | 名额分配到初中，本校内竞争 |
| **指标统调** | 2个平行学校志愿（A/B） | 指标到校未用完的名额回收，全市再分配 |

#### ⭐ 填写原则

**统招（3个平行志愿）**：
- 按"冲-稳-保"策略排列
- 志愿A填想冲的学校（高于自身成绩5-10分的学校）
- 志愿B填与自身成绩匹配的学校
- 志愿C填保底学校（低于自身成绩的学校）
- **平行志愿检索规则**：分数优先、遵循志愿、一轮投档

**指标到校（2个平行志愿）**：
- 只能填本校有指标名额的学校
- 在本初中内部竞争，录取线通常**低于统招线**
- 这是普通学生进名校的核心机会

**指标统调（2个平行志愿）**：
- 各初中指标到校未用完的名额回收后全市统一调配
- 录取分数通常介于统招线和指标到校线之间
- 相当于"指标到校的第二次机会"

### 第四段：一般普通高中
| 志愿栏 | 数量 | 说明 |
|--------|------|------|
| 平行学校志愿 | 6个（A~F） | 限报3所公办一般普通高中 + 3所民办普通高中/综合高中 |

**填写原则**：
- 公办和民办各限3所，根据自己意愿选择填报顺序
- 6个志愿全部为平行志愿
- 这是普通高中的兜底段，D段录不上就只能去职业院校

### 各列含义详解

#### 1. 统招分（统招录取分数线）
- **含义**：该高中在全市范围内统一招生的最低录取分数线
- **特点**：全市统一标准，不分初中学校，纯靠分数竞争
- **示例**：24中统招分777，意味着全大连市考777分及以上的学生可以通过统招进入24中
- **对应志愿表位置**：第二段 → 统招 → 3个平行志愿

#### 2. 指标数（指标到校名额）
- **含义**：该高中分配给9中的指标到校名额数量
- **特点**：每所初中分到的名额不同，由教育局按比例分配
- **示例**：24中分给9中6个名额，育明分给9中7个名额
- **对应志愿表位置**：第二段 → 指标到校 → 2个平行志愿

#### 3. 最高指标分
- **含义**：在9中校内，通过指标到校被该高中录取的学生中，**最高分是多少**
- **作用**：反映该校指标到校竞争的上限

#### 4. 最低指标分
- **含义**：在9中校内，通过指标到校被该高中录取的学生中，**最低分是多少**
- **作用**：这才是关键参考——**你在9中校内至少需要考多少分才能通过指标到校进入该高中**
- **示例**：24中最低指标分775，即9中学生考775分就能通过指标到校进24中

#### 5. 分差（最低指标分 - 统招分）
- **含义**：指标到校最低录取分与统招线的差值
- **正值**：指标到校最低分**高于**统招线（说明该校指标名额在9中竞争激烈，指标到校没有优势）
- **负值**：指标到校最低分**低于**统招线（说明通过指标到校可以**降分录取**，有优势）
- **零/小正值**：指标到校和统招差不多

---

### ⭐ 分差的实战意义

| 分差范围 | 含义 | 策略 |
|----------|------|------|
| **负值**（如24中 -0.5） | 指标到校可降分录取 | 值得用指标到校志愿报 |
| **0~3** | 指标到校和统招差不多 | 指标到校意义不大 |
| **3~8**（如8中 7, 辽附 5） | 指标到校竞争激烈，反而要求更高分 | 慎重，可能不如直接统招 |
| **≥8**（如20高 10, 2中 11） | 指标到校名额被高分学生占满 | 不建议靠指标到校，直接统招更合理 |

#### 典型案例分析

**24中**：统招777，指标最低775，分差 **-0.5**
→ 9中学生考775就能通过指标到校进24中，比统招线低2分，**指标到校有优势**

**8中**：统招767.5，指标最低767，分差 **7**（最高指标分774.5）
→ 虽然最低指标分和统招线接近，但指标名额被767-774.5分段的学生占满，竞争激烈

**20高**：统招751，指标最低749，分差 **10**（最高指标分761）
→ 指标到校最高分761远超统招线751，说明很多高分学生用指标到校来保底进20高

## 四、综合填报策略（以9中学生为例）

### 第二段填报思路

```
统招志愿A（冲）：填略高于自己分数的学校
统招志愿B（稳）：填与自己分数匹配的学校
统招志愿C（保）：填低于自己分数的学校

指标到校志愿A：选指标名额多（7个）、分差为负或较小的学校
指标到校志愿B：选第二优先的指标到校学校

指标统调志愿A/B：作为指标到校的补充
```

### 不同分数段策略示例

| 分数段 | 统招策略 | 指标到校策略 |
|--------|----------|------------|
| **775+** | 冲24中统招、稳育明、保8中 | 指标到校报24中（分差-0.5，有优势） |
| **765-775** | 冲育明、稳8中/辽附、保1中 | 指标到校报育明或8中 |
| **755-765** | 冲1中、稳23中/12中、保工附 | 指标到校报23中或12中 |
| **745-755** | 冲20高/36中、稳13中/3中、保48中 | 指标到校报20高（名额7个较多） |
| **730-745** | 冲11中/3中、稳16中/2中、保红旗 | 指标到校报11中或2中 |

### ⚠️ 重要提醒

1. **指标到校只在本校内竞争**：9中的数据只代表9中，其他初中的指标分完全不同
2. **平行志愿是一轮投档**：一旦被前面志愿录取，后面志愿自动作废
3. **指标到校需满足学籍条件**：需在本校连续就读且学籍一致
4. **2025数据仅供参考**：2026年的统招线和指标名额会有变化
5. **分差大的学校指标到校反而不划算**：高分学生用指标保底，会抬高指标到校最低分
